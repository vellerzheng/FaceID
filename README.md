
### 基于TensorFlow训练的人脸识别神经网络

### 用法：
* 安装tensorflow 开发环境
* pip install *** 安装基本库
* 电脑连接摄像头运行get_my_faces.py采集面部图片，采集一万张大概一个多小时。
* 下载他人的图片集运行set_other_faces训练，下载地址
 ![图片集下载:](http://vis-www.cs.umass.edu/lfw/lfw.tgz)

* 运行is_my_face.py 进行面部识别
